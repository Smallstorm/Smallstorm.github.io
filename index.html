Добро пожаловать на страницы моего блога!

На этой странице я решила представить пользователям все основные составляющие данного сайта и рассказать о его создании.

Набор языков, с помощью которых создавался ресурс довольно стандартный. Это HTML, CSS, немного Java Script и немного PHP. Важной частью оформления сайта стала библиотека инструментов Bootstrap.

Все начинается с Главной страницы. На ней вы сейчас и находитесь. Для оптимизации кодирования принято использовать управляющую конструкцию INCLUDE языка PHP. Что я и сделала. Это позволило мне подключить часть кода к основному файлу. Также для удобства восприятия весь код табулирован.

Основное наполнение сайта(контент) являются статьи. При их создании я использовала литературу данной тематики. А именно : Учебный курс для самостоятельной подготовки по Компьютерным сетям («Networking Essentials: Hands-On, Self-Paced Training for Supporting Local And Wide Area Networks»), Основы алгоритмизации и программирования В.Д. Колодаева. Для уточнения некоторых вопросов я пользовалась интернет ресурсами.
